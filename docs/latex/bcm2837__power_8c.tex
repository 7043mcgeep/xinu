\subsection{Detailed Description}
This driver provides the ability to power on and power off hardware, such as the U\-S\-B Controller, on the B\-C\-M2837\-B0 So\-C used on the Raspberry Pi 3 B+. This makes use of the B\-C\-M2837\-B0's mailbox mechanism.
